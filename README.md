# TEFA-sr
TEFA-SR（Texture Enhancement and Feature Adaptive Super-Resolution）是一种基于参考图像的超分辨率方法，通过引入高分辨率参考图像来恢复低分辨率图像的细节。该方法主要解决了以下三个关键问题： 分辨率差异导致的特征不匹配 通过Transformer架构，利用低分辨率特征作为查询（Query），参考图像特征作为键（Key）和值（Value），实现跨分辨率的纹理匹配。 高频细节提取困难 提出高频增强模块（HFEM），通过自适应池化和残差结构，有效提取并增强高频细节。 特征分布不一致 引入特征自适应模块（FAM），动态校准参考图像特征，使其与低分辨率图像特征分布一致，提升融合效果。
